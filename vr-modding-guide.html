PSVR2

PSVR2 modding

Not supported. PSVR2 is a fully locked platform. Sony does not allow sideloading, APK access, or code injection of any kind.

PSVR2 runs on a closed PlayStation ecosystem — there is no developer mode, no file system access, and no known method to mod IL2CPP games on the platform. This is unlikely to change.

If the game you want to mod has a PC VR version (e.g. on Steam), that would be your best alternative path.
